Hello There! (definitely a starwars reference)

This mini tiny project shows how to use NASA APOD API and provide easy fast output using streamlit.
"How to use it?
"First, install required imported libraries"
To test it out, sign in https://api.nasa.gov/ and get your API key for free
Replace the API_KEY with yours in secrects.toml and run it with the command below
streamlit run <file.py>

Customizations
 You can either use default streamlit panel settings or use config.toml

Checkout https://apod.nasa.gov/apod/archivepix.html to access the official archive
